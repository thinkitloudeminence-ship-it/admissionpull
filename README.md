# admissionPull.com

A premium, Awwwards-level animated website for educational institutions' growth platform.

## 🚀 Features

- **Three.js** animated 3D background with particle system
- **GSAP** scroll animations and text reveals
- **Lenis** smooth scrolling
- **Framer Motion** UI animations
- Glassmorphism design with gradient effects
- Fully responsive layout
- Custom cursor with interactive effects
- Animated counters for statistics
- 3D card tilt effects
- Contact form with floating labels

## 🛠️ Tech Stack

- Next.js 14 (App Router)
- React.js 18
- Tailwind CSS
- Three.js / React Three Fiber
- GSAP + ScrollTrigger
- Lenis
- Framer Motion
- React Icons
